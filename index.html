This site is expected to be a personal knowledge management.

Five Stages in Dreyfus Model

https://en.wikipedia.org/wiki/Dreyfus_model_of_skill_acquisition

Personal Knowledge Management

A piece of knowledge may be software systems, algorithms, data structures, philosophical ideas, etc. Based on the dreyfus model, I intend to divide the learning process of a knowledge into several stages.

  1. Find the getting started part of the official documentation.

  2. Follow tasks in different situations to practice. See official reference and API docs during practice.

  3. Understanding the principles and implementation details.

  4. Thinking about the overall system.

Content of a Knowledge

Input

  • Official Documents
  • Published Books
  • Blogs, Forum, Google, etc.

Output

  • URLs
  • Study Notes
  • Architecture Diagram, Design Diagram, Flow Chart
  • Sample Code
